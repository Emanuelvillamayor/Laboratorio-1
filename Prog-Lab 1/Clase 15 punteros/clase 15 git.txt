el git nos permite frenar nuestro codigo y luego siguieramos hacer la instantanea podes crear un zip
para eso se hace la instantanea tenemos que usar el comando : "git commit -m "primer commit"
Cada "commit" va a decir lo que vamos haciendo comentando y diciendo nuestros progresos.


puede ser que tenemos que estar logueados entonces ejecutamos esos comandos :
 git config --global user.email "you@example.com"
